# Supa Snake 🐍

Welcome to Supa Snake! A modern, responsive twist on the classic Snake game, brought to life with a vibrant retro-neon theme. Built with Next.js, Firebase, and Tailwind CSS, this project showcases a real-time, interactive web application.

## ✨ Features

- **Classic Gameplay, Modern Look**: Enjoy the timeless fun of Snake with a stunning neon aesthetic and smooth animations.
- **Custom Graphics**: Watch your snake, drawn with custom head and body segments, chase after cleverly designed rats instead of plain apples.
- **Progressive Difficulty**: The game gets faster and more challenging as you advance through levels by eating more food.
- **Real-time Global Leaderboard**: Compete against other players! The top 10 highest scores are displayed in real-time for everyone to see.
- **User Authentication**: Create an account or log in to save your personal high score to the global leaderboard.
- **Global Chat**: Once logged in, you can chat with other players in the global chat room.
- **Fully Responsive Design**: The game is designed to be played on any device.
  - **Mobile Layout**: On smaller screens, the leaderboard and chat are neatly organized into tabs.
  - **Desktop Layout**: On larger screens, components are displayed in a spacious side-by-side layout.
- **Immersive Fullscreen Mode**: Click the maximize icon to enter a distraction-free fullscreen mode for an enhanced gaming experience.
- **Device-Specific Controls**:
  - **Desktop**: Use the **Arrow Keys** to move, **Spacebar** to pause/resume, and the **'F' key** to toggle fullscreen.
  - **Mobile**: **Swipe** on the game screen to control the snake and use the dedicated on-screen button to pause and resume.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (with App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **Backend & Database**: [Firebase](https://firebase.google.com/) (Firestore for database, Authentication for users)
- **Deployment**: Ready for [Vercel](https://vercel.com/)

## 🎮 How to Play

1.  **Start the Game**: Press `Enter` or tap the screen to begin.
2.  **Control the Snake**:
    - On Desktop: Use the **Arrow Keys**.
    - On Mobile: **Swipe** in the direction you want to go.
3.  **Eat the Rats**: Guide the snake to eat the rats that appear on the screen.
4.  **Avoid Collisions**: Don't run into the walls or the snake's own tail!
5.  **Save Your Score**: Log in to have your high score automatically saved to the leaderboard.
